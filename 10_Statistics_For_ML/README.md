# Statistics for Machine Learning (Module 10)

Statistics is the core mathematical foundation of Machine Learning. It provides the framework for analyzing, summarizing, interpreting, and validating datasets to make data-driven decisions.

## Coding Format Rules for Statistics

Every python file contains:
- **Formula explanation** inside comment blocks.
- **Python implementation** using libraries like standard math, `numpy`, and `scipy.stats`.
- **Example datasets**.
- **Interpretation of results** printed directly to the console during execution.
