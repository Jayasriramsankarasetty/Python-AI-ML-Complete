# Generative AI (Module 15)

This module provides practical, executable implementations of Generative AI principles, Prompt Engineering paradigms, Vector Databases, and a complete Retrieval-Augmented Generation (RAG) pipeline from scratch.

---

## Module Roadmap

The module is partitioned into six key sections:

1. **[LLM_Basics/llm_basics.py](file:///c:/Users/S. Jaya Sri Ram/OneDrive/Desktop/Placements/Python-Journey/15_Generative_AI/LLM_Basics/llm_basics.py)**:
   * Covers API generation patterns, parameter settings (temperature, top_p), and custom model simulations.
2. **[Prompt_Engineering/prompts.py](file:///c:/Users/S. Jaya Sri Ram/OneDrive/Desktop/Placements/Python-Journey/15_Generative_AI/Prompt_Engineering/prompts.py)**:
   * Explains structures like Zero-Shot, Few-Shot, and Chain-of-Thought (CoT) reasoning templates.
3. **[Embeddings/embed_text.py](file:///c:/Users/S. Jaya Sri Ram/OneDrive/Desktop/Placements/Python-Journey/15_Generative_AI/Embeddings/embed_text.py)**:
   * Bridges text variables to continuous high-dimensional vector representations.
4. **[Vector_Databases/vector_db.py](file:///c:/Users/S. Jaya Sri Ram/OneDrive/Desktop/Placements/Python-Journey/15_Generative_AI/Vector_Databases/vector_db.py)**:
   * Simple in-memory nearest neighbors cosine search database engine.
5. **[LangChain/chains.py](file:///c:/Users/S. Jaya Sri Ram/OneDrive/Desktop/Placements/Python-Journey/15_Generative_AI/LangChain/chains.py)**:
   * Simulates sequencing prompts, output parsers, and memory state buffers.
6. **[RAG_Projects/rag_pipeline.py](file:///c:/Users/S. Jaya Sri Ram/OneDrive/Desktop/Placements/Python-Journey/15_Generative_AI/RAG_Projects/rag_pipeline.py)**:
   * Pre-packaged RAG implementation executing document loading, splitting, embedding indexing, query retrieval, and final augmented answer generation.
