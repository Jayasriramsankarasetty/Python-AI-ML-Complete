# Python AI/ML Placement Preparation Roadmap

Welcome to the ultimate preparation repository for Python, Data Structures & Algorithms (DSA), SQL, Statistics, Data Analysis, and Machine Learning engineering roles. This repository is structured sequentially to guide you from core programming concepts to advanced machine learning pipeline builds.

---

## Repository Structure

The curriculum is divided into 13 modules, each containing clean, self-contained, executable scripts:

### Part 1: Core Python Programming
1. **[01_Python_Basics/](01_Python_Basics/)**: Variables, operators, data types, and standard input/output.
2. **[02_Control_Flow/](02_Control_Flow/)**: Conditional checks, loops, and pattern programs.
3. **[03_Functions/](03_Functions/)**: Scopes, recursion, default parameters, and decorators.
4. **[04_Data_Structures/](04_Data_Structures/)**: Lists, tuples, sets, and dictionaries.
5. **[05_Object_Oriented_Programming/](05_Object_Oriented_Programming/)**: Classes, inheritance, polymorphism, and abstraction.
6. **[06_File_Exception_Modules/](06_File_Exception_Modules/)**: File handling, error catching, and custom modules.
7. **[07_Advanced_Python/](07_Advanced_Python/)**: Generators, iterators, list comprehensions, and lambda expressions.

### Part 2: Data Structures & Algorithms (DSA)
8. **[08_Python_DSA_For_Placements/](08_Python_DSA_For_Placements/)**: Focuses on Arrays, Linked Lists, Trees, Stacks, Queues, Sliding Window, Dynamic Programming, and Recursion.

### Part 3: Data Science & SQL
9. **[09_Python_Libraries_For_AI_ML/](09_Python_Libraries_For_AI_ML/)**: Pandas, NumPy, Matplotlib, and Seaborn libraries.
10. **[10_Statistics_For_ML/](10_Statistics_For_ML/)**: Probability distributions, Bayes' Theorem, Descriptive statistics, and Hypothesis tests.
11. **[11_Data_Analysis_Practice/](11_Data_Analysis_Practice/)**: Messy dataset cleaning, sales analysis projects, and employee churn analysis profiling.
12. **[12_SQL_For_Data_Roles/](12_SQL_For_Data_Roles/)**: SQLite SELECT projections, WHERE filtering, INNER/LEFT/SELF Joins, window functions, and interview queries.

### Part 4: Machine Learning Pipelines
13. **[13_Machine_Learning_With_Python/](13_Machine_Learning_With_Python/)**: Data Preprocessing, Supervised models, Unsupervised clustering, and Model grid search evaluation.

### Part 5: Deep Learning
14. **[14_Deep_Learning/](14_Deep_Learning/)**: Pure NumPy neural networks, PyTorch classification modules, and TensorFlow Keras classifiers.

### Part 6: Generative AI
15. **[15_Generative_AI/](15_Generative_AI/)**: Next token generation probabilities, Prompt engineering exemplars, Text embeddings, Vector databases, LangChain memory buffers, and RAG pipelines.

### Part 7: Machine Learning Deployment (MLOps)
16. **[16_ML_Deployment/](16_ML_Deployment/)**: REST API design concepts, FastAPI schema serving, Flask servers, Dockerfiles, GCP Artifact Registry auth, serverless Cloud Run deployments, automated GitHub Actions CI/CD workflows, and MLflow experiment log tracking.

### Part 8: AI/ML Practical Projects
17. **[17_AI_ML_Projects/](17_AI_ML_Projects/)**: E-commerce recommendation engines, churn indicators, credit card fraud detection, RAG chat assistants, and FastAPI serving microservice architectures.

### Part 9: Placement Preparation
18. **[18_Placement_Preparation/](18_Placement_Preparation/)**: Advanced Python, SQL, ML, and AI interview Q&A question sheets, multiple-choice questions, and quick-reference syntax sheets.

## Getting Started

1. Make sure you have Python 3.10+ installed.
2. Install all third-party libraries:
   ```bash
   pip install -r requirements.txt
   ```
3. Follow the 30-day curriculum timeline in **[roadmap.md](roadmap.md)**.
4. Run any file from the root directory:
   ```bash
   python 13_Machine_Learning_With_Python/Supervised_Learning/random_forest.py
   ```
